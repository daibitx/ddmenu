# 菜谱管理App - 前后端设计文档

## 一、项目概述

### 1.1 项目简介
一款面向个人及家庭的菜谱管理应用，支持菜谱创建、原料管理、饮食记录等功能。区分管理员和普通用户权限。

### 1.2 技术栈
- **前端**: Flutter (Dart)
- **后端**: RESTful API (Node.js/Python/Go等)
- **数据库**: PostgreSQL / MySQL
- **认证**: JWT Token

---

### 4.2 菜谱表 (recipes)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT PK | 菜谱ID |
| name | VARCHAR(100) | 菜名 |
| category_id | INT FK | 分类ID |
| image | VARCHAR(255) | 图片URL |
| cook_time | INT | 预计用时(分钟) |
| difficulty | TINYINT | 难度1-5 |
